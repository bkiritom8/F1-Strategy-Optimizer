# Apex Intelligence | F1 Strategy Frontend

Production React 19 + TypeScript dashboard for the F1 Strategy Optimizer. Visualizes real-time telemetry, historical trends, Monte Carlo simulations, and ML-powered strategy recommendations. Deployed on **Firebase Hosting**.

## Tech Stack

| Technology | Version | Purpose |
|---|---|---|
| React | 19 | UI framework |
| TypeScript | 5.x | Type safety |
| Vite | 6.x | Build tool + dev server |
| Tailwind CSS | 3.x | Utility-first styling |
| Zustand | — | Global state management |
| ApexCharts | — | Telemetry + lap data charts |
| Firebase Hosting | — | Production deployment |

## Design System (Glass-Dark)

Premium **Pro-Racing** aesthetic built around glassmorphism:

- **Design tokens**: `--glass-bg`, `--glass-border`, `--glass-blur` — consistent depth and blur throughout
- **Typography**: `Outfit` (bold headings for racing urgency) + `Inter` (body text for data readability) via Google Fonts
- **Color palette**: High-contrast racing accents over a dark base
- **Animations**: Circuit-accurate SVG track paths with kinetic F1 car animations and smooth entrance transitions

## Key Components

| File | Purpose |
|---|---|
| `views/LandingPage.tsx` | High-impact entry with glassmorphic stats, animated hero, driver/team cards |
| `views/RaceCommandCenter.tsx` | Real-time strategy dashboard — telemetry, lap charts, ML recommendations |
| `components/simulation/RaceSimulator.tsx` | 2D track map with animated car dots driven by SSE stream |
| `components/simulation/DynamicSimulationBackground.tsx` | Circuit-accurate SVG animation engine |
| `components/tracks/` | Per-circuit SVG path registry + renderer |
| `hooks/` | Custom React hooks (WebSocket, SSE, strategy polling) |
| `services/` | Typed API client — wraps all backend endpoints |
| `store/` | Zustand stores for race state, driver selections, simulation frames |

## Static Data

| File | Generated by |
|---|---|
| `public/data/car_performance.json` | `pipeline/scripts/build_car_performance.py` — year-aware constructor offsets (2018–2025) |

Regenerate after each new season completes:
```bash
python pipeline/scripts/build_car_performance.py \
  --input gs://f1optimizer-data-lake/processed/race_results.parquet \
  --output frontend/public/data/car_performance.json
```

## Development

### Prerequisites

- Node.js 18+
- npm

### Commands

```bash
npm install        # Install dependencies
npm run dev        # Dev server → http://localhost:3001
npm run build      # Production bundle → dist/
npm run preview    # Preview production build locally
npm test           # Run Vitest unit tests
npm run lint       # ESLint
```

## Deployment

Deployed automatically to **Firebase Hosting** via CI/CD on push to `main`.

Manual deploy:
```bash
npm run build
firebase deploy --only hosting
```
