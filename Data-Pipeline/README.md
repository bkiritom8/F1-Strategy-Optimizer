# Data-Pipeline

Legacy and protected pipeline artifacts for the F1 Strategy Optimizer.

> [!IMPORTANT]
> This directory contains critical historical data processing logic and archived artifacts. **DO NOT DELETE** or move files from this directory as they serve as the reference for current production pipelines.

## Contents

- **Processing Logic**: Historical scripts used for the initial 76-year data migration.
- **Reference Data**: Static lookups for driver IDs and team mappings across different eras.
- **Validation**: Legacy validation suites for bucket data integrity.

## Usage

This module is primarily for reference and auditing. New development should occur in the top-level `/pipeline` directory.
